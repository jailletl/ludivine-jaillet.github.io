Tous mes projets

  • Gestion des autorisations et des données utilisateurs

    Développement d'une application web interagissant avec une base de données globale pour gérer les accès aux zones restreintes. J'ai créé une solution conviviale pour les gestionnaires d'accès en utilisant Visual Studio, C#, ChartJS, Entity Framework, CRUD, SonarQube et GitHub. J'ai également progressé en HTML, CSS, JavaScript, SQL et participé à toutes les phases du développement, de l'identification des besoins à la publication, en optimisant et sécurisant l'application.

  • Synthèse des tickets de demande de support

    Développement d'une application web interagissant avec une base de données globale, représentant les différents stades de traitement des tickets, via des graphes dynamiques. J'ai conçu une interface graphique intégrant des objets dynamiques comme la récupération de photos et d'informations personnelles en utilisant Visual Studio, C#, SonarQube, GitHub, HTML, CSS, JavaScript et SQL.

  • Gestion et suivi des commandes

    Développement d'une application interne facilitant la gestion des différentes étapes des commandes clients interagissant avec la base de données commandes, basée sur Node-Red et SQL Server Management. Elle offre une vision détaillée de chaque étape de préparation et de configuration des solutions, rendant le processus plus rapide et convivial.
