Programmation

J'ai acquis des compétences dans la création de sites web en utilisant HTML, CSS et JavaScript, ainsi qu'en travaillant avec des frameworks pour le front-end. J'utilise également PHP et SQL pour gérer les tâches côté serveur. Mon expérience inclut également le développement d'applications web et mobiles en C# avec Visual Studio, ainsi que la gestion de bases de données avec SQL Server Management. J'ai aussi travaillé sur des projets en utilisant Node-Red pour le développement rapide d'applications et avec des outils comme GitHub pour le versioning et SonarQube pour l'analyse et l'optimisation du code.

Expériences Professionnelles

Lors de mon alternance chez Schneider Electric, j'ai acquis une expérience significative en développement d'applications web utilisant différentes technologies : Visual Studio (C#, HTML, CSS, JavaScript, ...), Node-Red pour le développement rapide d'applications, SQL Server Management pour la gestion des bases de données, GitHub pour gérer les versions et SonarQube (optimiser, analyser du code, détecter les bugs, les vulnérabilités) pour un code plus sécurisé et maintenable.

Expériences universitaires

Lors de mes études, j'ai utilisé Angular pour le développement front-end, travaillé avec des bases de données NoSQL, créé des serveurs API REST pour les applications web, développé des compétences en Java pour l'analyse syntaxique, utilisé WebGL pour la création de formes 3D et développé des applications mobiles avec Flutter.
